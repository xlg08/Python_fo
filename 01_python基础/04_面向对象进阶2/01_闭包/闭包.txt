闭包：
    用来延长函数内的变量的生命周期

闭包可以保存函数内的变量，而不会随着调用完函数而被销毁。

闭包：
    在函数嵌套的前提下，内部函数使用了外部函数的变量，并且外部函数返回了内部函数，
    1.在函数嵌套的前提下
        总结：实现闭包条件一：有嵌套
    2.内部函数使用了外部函数的变量        （形参也属于外部函数变量）
        总结：实现闭包条件一：有引用
    3.并且外部函数返回了内部函数
        总结：实现闭包条件一：有返回

闭包的构成条件：
    ①有嵌套：在函数嵌套(函数里面再定义函数)的前提下；
    ②有引用：内部函数使用了外部函数的变量(还包括外部函数的参数)；
    ③有返回：外部函数返回了内部函数名。



闭包：
    使用外部函数变量的内部函数

    # 外部函数
    def 外部函数名(外部参数):
        # 内部函数
        def 内部函数名(内部参数):
            ...[使用外部函数的变量]
        return 内部函数名      # 闭包
